Este projeto e um Backend escrito em Python com FastAPI, que busca servir como uma plataforma para um estabelecimento que funciona como Livraria/Tabletop Cafe.

O projeto esta sendo uma maneira de aprofundar meus conhecimentos em backend. Para implementa-lo, utilizei PostgreSQL, com o Framework FastAPI em Python, juntamente com Pydantic, SQLAlchemy, UV e Alembic.
